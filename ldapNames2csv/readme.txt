names.txt Contains Names as they are spelled in LDAP server.
Output is LDAP ldapEntries.csv in the Format of 
Name | username | email | employeeID


PHP.ini
++++++++++++++++++++++++++++++++++++++
enable extension=php_ldap.dll
++++++++++++++++++++++++++++++++++++++

runNames.bat has to be in the same folder with "ldapNames.php" and "names.txt"
